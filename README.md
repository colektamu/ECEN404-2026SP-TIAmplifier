The 2025-2026 TI Amplifier Capstone Project was a senior year capstone project under Texas A&M University and sponsored by Texas Instruments. The project was completed by Haotian Guan, Cole Keating, Braeden Murdock, and Jonathan Roman under the mentorship and supervision of Alex Anderson as a contact with Texas Instruments. The files included in this GitHub repository reflect the progress updates and final version of the project.

Each folder within the repository reflects a single subsystem that contributed to the final project. 
Braeden Murdock integrated pcb power subsystem contains files relevant to the power subsystem that powered the device. 
Cole - Pre and power amp contains the files relevant to the analog subsystem, both the preamp that modulated the signal and the power amp that amplified it into the speaker output. 
Microcontroller - Jonathan Roman contains the pcb and code files for the microcontroller subsystem, used to translate mobile app inputs into potentiometer states on the audio circuits.

All other files found in the repository without a set folder are related documentation such as the Functional System Requirements and Validation Plan.
